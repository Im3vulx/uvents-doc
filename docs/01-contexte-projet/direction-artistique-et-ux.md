# Direction artistique et UX

## Rôle de la direction artistique dans ONY

La direction artistique d’ONY ne constitue pas un simple habillage visuel.
Elle participe directement au positionnement produit, à la lisibilité de l’expérience et à la différenciation de la marque.

Le projet s’inscrit dans une logique :

- **mobile-first** ;
- **épurée** ;
- **énergique** ;
- **territoriale** ;
- **accessible** ;
- tournée vers un parcours utilisateur court entre **découverte** et **action**.

## Identité de marque

## 1. Le changement de nom : de Uvents à ONY

Le passage de **Uvents** à **ONY** répond à une stratégie de différenciation forte.

Le nom Uvents était jugé trop descriptif et trop proche du champ lexical des plateformes concurrentes centrées sur “events”.Le nom **ONY** permet au contraire :

- d’avoir une marque plus courte et plus mémorable ;
- de sortir du registre purement utilitaire ;
- de renforcer l’identité du projet ;
- de mieux porter un imaginaire de mouvement et d’impulsion.

## 2. Positionnement de marque

ONY cherche à se positionner comme un compagnon de découverte locale plus que comme une simple interface transactionnelle.

L’intention est de proposer :

- une expérience plus humaine ;
- une marque plus proche du terrain ;
- un ton moins froid que les outils concurrents ;
- une interface capable d’accompagner l’utilisateur sans l’écraser.

## Principes visuels

## 1. Palette et contraste

La direction visuelle repose sur une utilisation dominante du **orange ONY**, associé à des fonds sombres ou très contrastés afin de renforcer :

- la lisibilité ;
- la hiérarchie ;
- la présence de marque ;
- l’usage nomade en conditions de lumière variables.

## 2. Simplicité radicale

L’un des axes structurants du projet est la **simplicité radicale**.

Concrètement, cela implique :

- peu de friction ;
- peu de surcharge cognitive ;
- peu d’effets décoratifs inutiles ;
- un accès rapide à l’information utile ;
- une hiérarchie claire entre aperçu, détail et action.

## 3. Mobile-first

Le format mobile est prioritaire.

Le design doit donc favoriser :

- les formats verticaux ;
- les composants compacts ;
- les interactions lisibles au pouce ;
- les parcours courts ;
- les écrans facilement consultables en mouvement.

## Ton et discours

Le ton recherché pour ONY est :

- complice ;
- bienveillant ;
- inclusif ;
- accessible sans être enfantin ;
- rassurant sans être institutionnel.

La marque cherche à se différencier des interfaces événementielles trop froides ou trop purement transactionnelles.

## UX produit

## 1. Carte comme centre d’expérience

La carte interactive n’est pas un module secondaire.
Elle représente le cœur de l’expérience utilisateur.

Elle doit permettre :

- une découverte immédiate ;
- une lecture territoriale des événements ;
- une cohérence entre proximité et pertinence ;
- un passage rapide du repérage à l’action.

## 2. Découverte → Résumé → Détail → Action

Le produit suit une logique d’UX progressive :

1. découverte rapide via liste ou carte ;
2. résumé lisible via cards et overlays ;
3. accès au détail complet ;
4. action éventuelle : favori, billet, scan, navigation.

Cette hiérarchie a guidé une grande partie des derniers travaux de refonte UI.

## 3. Cohérence d’interface

Les écrans doivent partager un langage commun :

- cartes arrondies ;
- contrastes marqués ;
- CTA visibles mais compacts ;
- overlays propres ;
- transitions simples ;
- bottom bar lisible ;
- hiérarchie stable d’un écran à l’autre.

## Accessibilité

La direction artistique est contrainte par une exigence forte d’accessibilité.

Cela implique :

- respect des contrastes ;
- lisibilité sur mobile ;
- tailles de texte cohérentes ;
- hiérarchie claire ;
- limitation du bruit visuel ;
- parcours compréhensibles même sur écran réduit.

L’accessibilité ne doit pas être traitée comme un correctif tardif, mais comme un cadre structurant du design.

## Conséquences sur l’implémentation

Dans l’interface ONY, la direction artistique se traduit par :

- une dominance orange / noir / blanc ;
- des composants compacts et lisibles ;
- des overlays plus informatifs qu’intrusifs ;
- des parcours simplifiés ;
- une attention particulière portée à la map ;
- un travail d’harmonisation de plus en plus poussé entre les écrans.

## État actuel

La plupart des écrans majeurs ont déjà été retravaillés dans cette logique :

- authentification ;
- accueil ;
- page events ;
- map ;
- filtres ;
- overlays événements ;
- billets ;
- profil ;
- détail événement.

Le cap actuel n’est donc plus de définir la direction artistique, mais de la **documenter**, la **maintenir** et l’**appliquer de façon cohérente** sur les zones encore moins abouties.

## Ligne directrice pour la suite

Pour toutes les futures évolutions UI/UX, les décisions devront rester cohérentes avec les principes suivants :

- la carte reste le centre de gravité du produit ;
- le mobile reste le format prioritaire ;
- la lecture doit rester rapide et intuitive ;
- la marque doit rester identifiable ;
- l’accessibilité doit rester un critère de décision ;
- la simplicité prime sur la sophistication inutile.
