# Liens utiles

## Objectif de cette section

Cette page centralise les principaux **liens utiles** liés au projet **ONY**.

L’objectif est de regrouper en un seul endroit les accès les plus importants pour :

- la documentation ;
- le dépôt applicatif ;
- le dépôt documentaire ;
- les ressources techniques structurantes du projet.

Cette page a vocation à servir de point d’entrée pratique pour retrouver rapidement les ressources essentielles.

## Documentation

### ONY Docs

Documentation technique, fonctionnelle et d’exploitation du projet.

- Documentation publique : `https://uvents-doc-5a21db.gitlab.io`

## Dépôts GitLab

### Dépôt applicatif

Dépôt principal contenant le code de l’application ONY.

- GitLab App : `https://gitlab.com/julien-lesimple/uvents-app`

### Dépôt documentaire

Dépôt principal contenant la documentation Docusaurus du projet.

- GitLab Doc : `https://gitlab.com/julien-lesimple/uvents-doc`

## Navigation documentaire

La configuration actuelle de la documentation prévoit des accès rapides vers les grandes sections du projet, notamment :

- Contexte
- Architecture
- Application
- Infrastructure
- Exploitation

Cette organisation permet de retrouver rapidement les familles de documents les plus importantes.

## Ressources projet à centraliser

Selon l’évolution du projet, cette page peut aussi accueillir d’autres liens utiles, par exemple :

- environnement de préproduction ;
- environnement de production ;
- console d’administration d’un service externe ;
- dashboard Supabase ;
- dashboard Stripe ;
- espace de monitoring ;
- board projet ;
- documentation annexe ;
- outils de recette ou de test.

L’objectif reste de garder une page concise, orientée usage réel, sans la transformer en inventaire confus.

## À retenir

Cette page sert de point d’accès rapide aux ressources importantes d’ONY.

Dans son état minimal, elle doit au moins regrouper :

- la documentation ;
- le dépôt applicatif ;
- le dépôt documentaire.

Elle pourra être enrichie progressivement selon les besoins du projet.
